# 🏛️ Architecture Alpha Omega : Système Multi-Pilotes

## 🎯 Vision du Concepteur
Cette architecture rejette les systèmes monolithiques traditionnels au profit d'une **Intelligence Décentralisée**. L'objectif est de segmenter les responsabilités pour garantir une résilience maximale : le code gère la sécurité immuable, tandis que l'IA gère l'adaptation stratégique.

## 🏗️ Points Forts de l'Orchestration
* **Déterminisme Sécurisé** : Un cœur en Node.js gère le "Verrou de Réserve" (protection du capital) sans interférence de l'IA.
* **Modularité "Satellite"** : Chaque capteur est indépendant, permettant d'ajouter ou de retirer des métriques d'analyse sans compromettre le système.
* **Pilotage Adaptatif** : Utilisation de modèles Gemini pour ajuster les seuils de décision en temps réel selon la volatilité du marché.
